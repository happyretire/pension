# 🏛️ 공무원 퇴직연금 개시 연령 계산기

> **"나의 정확한 연금 수령 시기를 한눈에 확인하세요."**  
> 복잡한 공무원 연금 지급 개시 연령을 임용 시기와 퇴직 연도에 맞춰 정밀하게 산출해 드립니다.

---

## 🌟 주요 특징

### 1. 정밀한 법적 로직 반영

- **임용 시기별 구분**: 1995년 12월 31일 이전/이후 임용자에 대한 각기 다른 법적 기준을 완벽하게 적용합니다.
- **경과 규정 완벽 지원**: 2000년 기준 재직기간에 따른 특례 조항(재직 20년 도달 시 즉시 수령 등)을 반영하여 연령 도달 전 수령 케이스까지 계산합니다.
- **퇴직 연도별 연령 매핑**: 2016년부터 2033년 이후까지 단계적으로 상향되는 60~65세 지급 연령을 자동으로 매핑합니다.

### 2. 구체적인 수령 시점 계산

- 단순히 '수령 연령'만 보여주는 것이 아니라, 생년월일을 기반으로 **실제 수령 예상 연월(생일 다음 달)**을 정확히 산출합니다.
- 이미 연령 요건을 갖춘 상태에서 퇴직하는 경우, **퇴직 다음 달** 수수령 시기를 구체적으로 계산하여 안내합니다.

### 3. 프리미엄 사용자 인터페이스 (UI)

- **Glassmorphism 디자인**: 세련된 유리 질감의 다크 모드 UI와 생동감 넘치는 그라데이션 배경을 제공합니다.
- **반응형 최적화**: PC, 태블릿, 스마트폰 등 어떤 기기에서도 최적화된 레이아웃으로 편리하게 사용할 수 있습니다.
- **스마트 유효성 검사**: 날짜 입력 순서 오류를 사전에 방지하고, 직관적인 결과 애니메이션을 제공합니다.

---

## 🛠️ 사용 방법

1. **파일 실행**: `index.html` 파일을 웹 브라우저(Chrome, Edge, Safari 등)에서 더블 클릭하여 엽니다.
2. **정보 입력**:
    - **생년월일**: 본인의 생년월일을 입력합니다.
    - **공무원 임용일**: 실제 임용(발령)되어 근무를 시작한 날짜를 입력합니다.
    - **퇴직 예상 연월일**: 퇴직 예정일을 선택합니다. (오늘 이후의 날짜만 선택 가능합니다.)
3. **결과 확인**: `계산 결과 확인` 버튼을 누르면 하단에 즉시 **지급 개시 연령**과 **상세 연월** 결과가 나타납니다.

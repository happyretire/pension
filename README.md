# 공무원 퇴직연금 개시 연령 계산기

공무원 임용 시기와 퇴직 연도에 따른 연금 지급 개시 연령을 자동으로 계산해주는 프로그램입니다.

## 주요 기능
- 임용일자별 맞춤 로직 적용 (1995년 이전/이후)
- 퇴직 연도별 지급 연령 자동 매핑
- 생년월일 기준 예상 지급 개시 연도 산출
- 현대적이고 세련된 Glassmorphism 디자인

## 사용 방법
1. `index.html` 파일을 웹 브라우저(Chrome, Edge 등)로 엽니다.
2. 임용일, 생년월일, 퇴직(예정) 연도를 입력합니다.
3. '계산하기' 버튼을 클릭하여 결과를 확인합니다.

# 🏦 공무원 퇴직연금 개시일 계산기

임용 시기와 퇴직 연도에 따른 공무원 퇴직연금 지급 개시 시점을 정밀하게 계산해 주는 웹 어플리케이션입니다.

---

## ✨ 주요 기능

* **📅 맞춤형 개시 연령 산출:** 사용자의 생년월일과 퇴직 연도를 바탕으로 법정 연금 개시 연령을 자동으로 판정합니다.
* **⚖️ 1996년 이전 임용자 특례 반영:** 1996년 이전에 임용된 공무원을 위한 복잡한 특례 계산 로직(부족 기간의 2배 가산 등)이 완벽하게 반영되어 있습니다.
* **⏳ 실근무 기간 자동 계산:** 휴직 기간(연금법상 재직기간 미포함분)을 입력하면 실근무 기간을 자동으로 계산하여 수급 자격을 판정합니다.
* **✅ 연금 수급 자격 체크:** 2016년 개정안에 따라 퇴직 시점별 수급 자격(10년 또는 20년) 충족 여부를 즉시 알려줍니다.
* **📱 프리미엄 글래스모피즘 디자인:** Lucide 아이콘과 세련된 다크 모드 UI를 적용하여 모던하고 쾌적한 사용자 경험을 제공합니다.

---

## 🚀 사용방법

1. **생년월일 입력:** 본인의 생년월일을 선택합니다.
2. **공무원 임용일 입력:** 공직에 처음 임용한 날짜를 선택합니다.
3. **퇴직 (예상) 연월일 입력:** 실제 퇴직했거나 퇴직 예정인 날짜를 선택합니다.
4. **휴직 기간 설정:** 연금법상 재직기간에 산입되지 않는 휴직 기간이 있다면 개월 단위로 입력합니다.
5. **계산 결과 확인:** 버튼을 클릭하면 '수급 가능 시기'와 '실제 첫 수령 시기'를 확인할 수 있습니다.
